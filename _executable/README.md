# 주의사항

## 브라우저 드라이버의 경로는 C:\Scrapper\browserDriver(환경변수 경로) 로 설정되어야만 합니다

## setEnvVariable.bat(환경변수 설정)을 실행시켜 브라우저 드라이버가 있는 경로를 환경변수에 추가시켜야 합니다

## 위의 배치파일은 관리자 권한으로 실행되어야 합니다
